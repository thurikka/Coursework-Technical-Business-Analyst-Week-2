# Docs folder

This folder contains the Week 2 written guides and structure templates.

## Files in this folder

- `01-phase-1-scope-template.md` - helps you define what belongs in the first release and what does not
- `02-backlog-quality-guide.md` - explains how to write stronger stories and acceptance criteria
- `03-wireframe-annotation-guide.md` - shows how to annotate low-fidelity screens for review
- `04-executive-briefing-storyboard.md` - slide-by-slide executive briefing template

## Recommended reading order

1. Scope template
2. Backlog guide
3. Wireframe guide
4. Executive storyboard

## Tip

If your work starts to feel disconnected, come back here and follow the order again. Week 2 works best when scope drives backlog, backlog drives prototype, and all three support the executive case.
