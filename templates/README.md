# Templates folder

This folder contains reusable Week 2 working templates.

## Files in this folder

- `adkar-assessment-template.csv` - starter stakeholder change-risk analysis table
- `jira-backlog-template.csv` - backlog starter with example story rows
- `figma-screen-inventory-template.csv` - traceability sheet between screens and stories
- `executive-briefing-speaker-notes-template.md` - support for slide narration and message discipline

## How to use these templates

Populate the CSVs in Excel or Google Sheets, then import into your preferred tools if needed.

## Reminder

These are intentionally incomplete. Your job is to fill the analysis and delivery logic, not just tidy formatting.
