# Data folder

This folder contains Week 2 reference inputs that connect discovery evidence to execution-ready outputs.

## Files in this folder

- `phase1_opportunity_ranking_reference.csv` - starter ranking table based on a plausible Week 1 outcome
- `story_to_screen_traceability_reference.csv` - example mapping between backlog stories and prototype screens

## How to use these files

Use them if you do not yet have your own finished Week 1 outputs. If you do have Week 1 outputs, replace these starter values with your own evidence.
